# Introduction to Physical AI & Humanoid Robotics

Welcome to the comprehensive textbook on Physical AI and Humanoid Robotics. This textbook provides hands-on mastery of modern humanoid robotics through simulation-first approaches with optional hardware extensions.

## About This Textbook

This textbook is designed to guide you through 13 weeks of intensive learning covering:

- **Module 1: Foundations** - Understanding Physical AI, humanoid robotics fundamentals, and the ROS 2 ecosystem
- **Module 2: Simulation Environments** - Working with both Gazebo and NVIDIA Isaac Sim for advanced humanoid simulation
- **Module 3: Perception and Control** - Sensor integration, computer vision, and control systems
- **Module 4: Vision-Language-Action Integration** - Advanced VLA models and human-robot interaction

## Target Audience

This textbook is designed for:
- Graduate students in robotics and AI
- Researchers working on embodied intelligence
- Engineers developing humanoid robots
- Developers interested in simulation environments

## Prerequisites

Before starting this textbook, you should have:
- Basic knowledge of Python programming
- Understanding of linear algebra and calculus
- Familiarity with Linux command line
- Basic understanding of robotics concepts

## Learning Approach

This textbook emphasizes:
- **Simulation-first approach** with optional hardware extensions
- **Hands-on mastery** with 10+ runnable code examples
- **Practical hardware guidance** for real-world applications
- **Exercises** to reinforce concepts

## Technical Stack

Throughout this textbook, we'll be using:
- **ROS 2 Jazzy/Kilted Kaiju** for the robotics framework
- **NVIDIA Isaac Sim 5.0+** for advanced humanoid simulation
- **Gazebo Garden/Harmonic** for open-source simulation
- **Python 3.11+** with rclpy for ROS 2 client library
- **Docusaurus** for interactive documentation with MDX
